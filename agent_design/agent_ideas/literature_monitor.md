### Komodo: The Literature Monitor - 

- runs autonomously to find the latest articles relevant to the user's goals
- writes brief summaries of the most interesting and presenting the summaries to the user. 
- maintains a model of the articles that it has found
    - Authors
    - Citations
    - Whether the user has read them
- Follows citations to find prior articles and processing them as well
- Can do full article processing
- Maintains a model of the completeness and adequacy of its current database of articles.
- responds to user questions and requests
    - ask about papers already processed
    - ask for full text processing
    - ask for one-time research project
    - modify the user's goals
